cobra 快速的实现一个命令行客户端
1.命令行项目的三个关键概念
- 命令：代表要做什么事情，代表动作，比如说 git add,git status 的 add 和 status 是子命令，git 是 root 命令
- 参数(args)：代表事物。git add go.mod 中的 go.mod 就是命令行的参数
- 标志(flags)：代表动作的修饰。git add --help 的 --help 就是标志
    - 本地标志，只能在当前命令使用
    - 持久化标志，当前命令及当前命令的所有下级命令都可以使用
        - 全局标志，根命令的持久化标志，所有命令都能使用
2.flags 的定义与配置绑定
- viper对象的值可以从 命令行里获取，也可以从配置文件里获取
3.自定义参数验证与内置验证
